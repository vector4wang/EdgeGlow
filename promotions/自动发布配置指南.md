## 🔧 方案一：MCP Server 安装（推荐）

### 1. 小红书 MCP (xiaohongshu-mcp)

**GitHub**: https://github.com/xpzouying/xiaohongshu-mcp

**安装步骤**:

```bash
# 1. 克隆仓库
cd ~/code
git clone https://github.com/xpzouying/xiaohongshu-mcp.git
cd xiaohongshu-mcp

# 2. 安装依赖
npm install

# 3. 构建
npm run build

# 4. 配置 MCP Server
# 在 ~/.claude/settings.json 的 mcpServers 中添加：
```

**配置示例** (添加到 `~/.claude/settings.json`):

```json
{
  "mcpServers": {
    "xiaohongshu": {
      "command": "node",
      "args": [
        "/Users/wangxingchao/code/xiaohongshu-mcp/dist/index.js"
      ],
      "env": {
        "XHS_COOKIE": "你的小红书Cookie"
      }
    }
  }
}
```

**获取 Cookie**:
1. 打开小红书网页版：https://www.xiaohongshu.com
2. 登录账号
3. 打开浏览器开发者工具 (F12)
4. 切换到 Network 标签
5. 刷新页面，找到任意请求
6. 复制 Request Headers 中的 Cookie 值

### 2. 微信公众号 MCP (wenyan-mcp)

**介绍**: https://zhuanlan.zhihu.com/p/1913645972132730250

**安装步骤**:

```bash
# 1. 克隆仓库（需要先找到 GitHub 地址）
cd ~/code
git clone <wenyan-mcp-repo-url>
cd wenyan-mcp

# 2. 安装依赖
npm install

# 3. 构建
npm run build

# 4. 配置 MCP Server
```

**配置示例**:

```json
{
  "mcpServers": {
    "wenyan": {
      "command": "node",
      "args": [
        "/Users/wangxingchao/code/wenyan-mcp/dist/index.js"
      ],
      "env": {
        "WECHAT_APPID": "你的公众号AppID",
        "WECHAT_SECRET": "你的公众号Secret"
      }
    }
  }
}
```

**获取公众号凭证**:
1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入「设置与开发」→「基本配置」
3. 获取 AppID 和 AppSecret

### 3. WechatSync（多平台同步）

**GitHub**: https://github.com/wechatsync/Wechatsync  
**官网**: https://www.wechatsync.com

**安装步骤**:

```bash
# 1. 安装 Chrome 扩展
# 访问 Chrome Web Store 搜索 "WechatSync" 或 "文章同步助手"

# 2. 或者克隆仓库本地开发
cd ~/code
git clone https://github.com/wechatsync/Wechatsync.git
cd Wechatsync

# 3. 安装依赖
npm install

# 4. 构建
npm run build

# 5. 在 Chrome 中加载扩展
# chrome://extensions/ → 开发者模式 → 加载已解压的扩展程序
```

**配置 MCP** (已支持 MCP 协议):

```json
{
  "mcpServers": {
    "wechatsync": {
      "command": "node",
      "args": [
        "/Users/wangxingchao/code/Wechatsync/mcp-server/index.js"
      ]
    }
  }
}
```

**支持平台**: 29+ 平台，包括知乎、掘金、CSDN、头条、百家号等

## 🔍 故障排查

### Git 克隆失败

**问题**: `fatal: unable to access`

**解决**:
```bash
# 1. 检查网络连接
ping github.com

# 2. 使用代理（如果需要）
export https_proxy=http://127.0.0.1:7890
export http_proxy=http://127.0.0.1:7890

# 3. 或使用镜像
git clone https://ghproxy.com/https://github.com/xpzouying/xiaohongshu-mcp.git
```

### npm 安装失败

**问题**: `npm install` 报错

**解决**:
```bash
# 1. 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 2. 清理缓存
npm cache clean --force

# 3. 重新安装
rm -rf node_modules package-lock.json
npm install
```

### MCP Server 启动失败

**问题**: Claude Code 无法连接 MCP Server

**解决**:
```bash
# 1. 检查配置文件
cat ~/.claude/settings.json | jq .mcpServers

# 2. 手动测试启动
node /path/to/mcp-server/dist/index.js

# 3. 检查端口占用
lsof -i :端口号

# 4. 重启 Claude Code
```
