### 回复模板

**Q: 怎么用？**
```
下载后打开 → 设置 → 配置 Agent Hooks → 复制引导词 → 发给 Claude Code → 搞定✨
```

**Q: 支持 Windows 吗？**
```
目前只支持 macOS 哦🥺 因为是原生 SwiftUI 应用
```

**Q: 会不会很卡？**
```
完全不会！CPU 占用几乎为 0，内存才 50MB，比浏览器还轻量🫶
```

**Q: 安全吗？**
```
绝对安全！只监听本地 127.0.0.1，不联网，不收集数据，开源代码可查✅
```

**Q: 和其他工具有什么区别？**
```
这个是专门为 AI 编程设计的，通过 HTTP hooks 和 Claude Code 联动，不是通用的屏幕特效工具哦✨
```
