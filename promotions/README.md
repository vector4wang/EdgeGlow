## 💡 通用发布策略

### 时间选择

**国际平台**
- 最佳：周二/周三/周四 09:00-11:00 PT
- 避免：周一、周五、周末

**中文平台**
- 技术社区：周二/周三/周四 10:00-12:00
- 生活方式：周二/周三/周四 20:00-22:00
- 视频平台：周五/周六/周日 18:00-20:00

### 互动要点

1. **发布后 1-2 小时**：必须在线，快速回复评论
2. **回复风格**：
   - 国际：技术导向、简洁、不推销
   - 中文：友好、详细、感谢反馈
3. **不要**：
   - 要求朋友刷票
   - 使用自动化工具
   - 过度营销
   - 贬低其他工具

### 内容调整

根据平台反馈，可以：
- 调整标题和封面
- 修改技术深度
- 增加或删除代码示例
- 调整语气和风格
