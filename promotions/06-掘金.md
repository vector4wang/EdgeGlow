## 正文

### 前言

用 Claude Code 写代码的时候，有个痛点：**你永远不知道 AI 是在思考还是在等你操作**。经常 alt-tab 切过去看，打断自己的心流。

所以我做了 **EdgeGlow** —— 一个 macOS 菜单栏小工具，AI 工作时屏幕边缘会亮起流光效果，完成了就自动消失。

> 从此，**AI 的工作状态一目了然**。余光一瞥，就知道该继续等待还是可以操作。

### 工作原理

很简单：

1. Claude Code 的 hooks 配置会在特定事件触发时调用 `curl http://127.0.0.1:9876/start`
2. EdgeGlow 内置的 HTTP 服务器收到请求后，触发流光动画
3. 收到 `/stop` 后，流光淡出

```
AI Agent 执行操作
       │
       ▼
  触发 Hook（Claude Code 配置的事件）
       │
       ▼
  curl http://127.0.0.1:9876/start
       │
       ▼
  ControlServer 解析 HTTP 请求
       │
       ▼
  GlowWindow 切换状态：
       ├── /start  → 重建图层 + 启动 Timer + 淡入
       ├── /pulse  → 停止 Timer + 保持静态显示
       └── /stop   → 停止 Timer + 淡出（1.5s）
```

### 技术亮点

#### 1. 四层光效模拟霓虹灯

为了做出真实的霓虹灯效果，我用了 4 层 `CAShapeLayer` 叠加：

```swift
let configs: [(widthMul: CGFloat, alphaMul: CGFloat, blur: Double)] = [
    (baseWidth * 1.5, 0.15, 12.0),  // 外层：宽线 + 高模糊 + 低透明度
    (baseWidth * 0.8, 0.30, 8.0),   // 中层：中线 + 中模糊 + 中透明度
    (baseWidth * 0.3, 0.70, 2.0),   // 内层：细线 + 低模糊 + 高透明度
    (baseWidth * 0.1, 0.95, 0.0),   // 核心：最细 + 无模糊 + 全透明度
]
```

```
                  屏幕边缘
                     │
    ┌────────────────┼────────────────┐
    │   Layer 1      │  blur=12       │  大光晕 (最外层)
    │   ████████     │  width=1.5x    │  alpha=0.15
    │                │                │
    │   Layer 2      │  blur=8        │  中光晕
    │   ██████       │  width=0.8x    │  alpha=0.30
    │                │                │
    │   Layer 3      │  blur=2        │  主色线 (最亮)
    │   ████         │  width=0.3x    │  alpha=0.70
    │                │                │
    │   Layer 4      │  blur=0        │  亮芯 (最锐利)
    │   ██           │  width=0.1x    │  alpha=0.95
    └────────────────┴────────────────┘
```

四层叠加 = 真实的霓虹灯效果。

#### 2. Timer 驱动动画（不依赖 Core Animation）

一开始用 `CABasicAnimation` 做 `lineDashPhase` 动画：

```swift
let anim = CABasicAnimation(keyPath: "lineDashPhase")
anim.fromValue = 0
anim.toValue = perimeter
anim.repeatCount = .infinity
shape.add(anim, forKey: "flow")
```

但窗口隐藏再显示后，动画会卡住甚至反转。

**改用 Timer 驱动**：

```swift
flowTimer = Timer.scheduledTimer(withTimeInterval: 1.0/60.0, repeats: true) { [weak self] _ in
    let now = CACurrentMediaTime()
    let dt = CGFloat(now - lastTickTime)
    lastTickTime = now
    let clampedDt = min(dt, 0.1)  // 防止从后台回来时跳变
    
    let speed = perimeter / CGFloat(animationDuration)
    dashPhase += speed * clampedDt * (clockwise ? 1 : -1)
    
    ringLayer.sublayers?.forEach { layer in
        (layer as? CAShapeLayer)?.lineDashPhase = dashPhase
    }
}
```

不依赖 Core Animation 的动画系统，窗口怎么切换都不会出问题。

#### 3. 多终端引用计数

多个 Claude Code 终端同时跑？一个 `/stop` 不会灭掉流光 —— 用引用计数，全部停了才灭。

```swift
class ControlServer {
    private var activeCount = 0
    
    case "/start":
        activeCount += 1
        onStart?()
        
    case "/stop":
        activeCount = max(0, activeCount - 1)
        if activeCount == 0 { onStop?() }
}
```

还有 60 秒安全超时：如果 Agent 崩溃没发 `/stop`，60 秒后自动归零。

#### 4. 安全设计

- HTTP 服务仅绑定 `127.0.0.1`（`acceptLocalOnly = true`）
- 仅接受 `GET` 请求，拒绝 POST/PUT/DELETE
- 无 CORS 头，网页 JavaScript 无法直接调用
- 不收集任何用户数据，无遥测

```swift
let param = NWParameters.tcp
param.acceptLocalOnly = true  // 仅接受本地连接
```

### 配置 Agent 联动

打开设置 → 点击「配置 Agent Hooks」→ 选择「Claude Code 查看配置」→ 复制引导词。

将引导词粘贴到 Claude Code 对话框中发送，Agent 会自动帮你完成 `~/.claude/settings.json` 的配置。

<details>
<summary>📋 手动配置参考</summary>

在 `~/.claude/settings.json` 中添加：

```json
{
  "hooks": {
    "UserPromptSubmit":  [{"hooks": [{"type": "command", "command": "curl -s http://127.0.0.1:9876/start"}]}],
    "PreToolUse":        [{"hooks": [{"type": "command", "command": "curl -s http://127.0.0.1:9876/start"}]}],
    "PostToolUse":       [{"hooks": [{"type": "command", "command": "curl -s http://127.0.0.1:9876/pulse"}]}],
    "Stop":              [{"hooks": [{"type": "command", "command": "curl -s http://127.0.0.1:9876/stop"}]}]
  }
}
```

</details>

### 安装使用

> **系统要求：macOS 13.0 (Ventura) 及以上**

#### 方式一：下载 Release（推荐）

前往 [Releases](https://github.com/vector4wang/EdgeGlow/releases) 下载最新 `EdgeGlow-v1.3.0.zip`，解压后打开。

> ⚠️ 首次打开如遇「无法验证开发者」提示，请 **右键 → 打开** 即可。

#### 方式二：源码编译

```bash
git clone https://github.com/vector4wang/EdgeGlow.git
cd EdgeGlow

# 编译（自动构建 Universal Binary: arm64 + x86_64）
./build.sh

# 运行
open Build/EdgeGlow.app
```
